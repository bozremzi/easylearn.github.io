Komutlar Açıklama
hostname "NAME" Router'ın adını değiştirir.
enable Ayrıcalıklı exec kipine geçer.
configure terminal Yapılandırma moduna geçer.
show COMMAND Yapılandırma hakkında bilgiler verir.
no shutdown Kapalı portu açmak için kullanılır.
ip dhcp pool "NAME" Router'a DHCP havuzu ismi verir.
clock rate "NUMBER" İki router serial port ile bağlandıysa clock rate belirlemek için Router'ın birine bu komut yazılır.
clear ip route Router'ın tuttuğu Routing Table bilgilerini temizler.
enable secret "PASSWORD" Router'a verilen şifreyi cryiptolar.
login Console'a bağlanırken parola ister.
banner motd #MESSAGE# Cihaz açıklarken kullanıcıya açıklama mesajı gösterir.
ip route Static ip tanımlamak için kullanılır.
service password-encryption Cihazda ayarlanmış/yapılandırılmış tüm mesajları cryptolar.
copy running-config startup-config Yapılan konfigurasyonları Router'ın NVRAM'i üzerine (Kalıcı) kaydeder.
interface PORT NAME PORT NUMBER Port yapılandırma arayüzüne gider.
