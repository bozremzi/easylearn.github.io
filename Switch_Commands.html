Komutlar Açıklama
hostname "NAME" Switch'in adını değiştirir.
enable Ayrıcalıklı exec kipine geçer.
configure terminal Yapılandırma moduna geçer.
show COMMAND Yapılandırma hakkında bilgiler verir.
switchport access vlan "NUMBER" Switch'in bir portunu VLAN'a atamak için kullanılır.
vlan "NUMBER" VLAN oluşturur.
switchport mode trunk İki Switch arasında belirlenen portu Trunk yapar.
exit O anki configuration modundan çıkmak için kullanılır.
enable secret "PASSWORD" Switch'e verilen şifreyi cryiptolar.
line vty 0 4 Telnet ile giriş yapmak için yapılandırılması gereklidir.
banner motd #MESSAGE# MOTD: (Message of the day) Günün Mesajını ayarlar.
interface vlan "NUMBER" Girilen VLAN arayüzü için yapılandırma moduna geçer.
speed auto Port hızını otomatik ayarlar.
copy running-config startup-config Yapılan konfigurasyonları kaydeder.
ip http authentication Yerel kullanıcı veri tabanını kullanmak istediğinde HTTP kimlik doğrulamasını etkinleştirir.
